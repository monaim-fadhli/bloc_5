Bienvenue sur la documentation de mon site web et mon API

Cette API permet de predire les locations de voitures, soit via une IHM (formulaire) ou bien via des call API

Base url

https://bloc-5.onrender.com

1.Post /

Description : ce point de terminaison permet predire une location de voiture (risqué ou non ) en se basant sur des données fournies en json (API) ou bien saisie directement par l'utilisateur en utilisant le formulaire

Requete

Methode: post

URL: https://bloc-5.onrender.com/

En-tete:

  • "Content-Type" = "application/json"

Corps json attendu

{
    "checking type" : 0,
    "engine_power_class" : 0,
    "paint_color" : 5,
    "has_gps" : 1,
    "rental_price_per_day": 122
}

Reponse (succes)

    {
        Prédiction: valeur prédit
    }

Reponse (échec)

    {
        error: invalid json
    }
